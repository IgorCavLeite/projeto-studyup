# 🚀 StudyUp - Gerenciador de Estudos

O **StudyUp** é uma ferramenta desenvolvida para auxiliar estudantes na organização de ciclos de estudo, utilizando a técnica Pomodoro e revisões baseadas em desempenho.

## 🛠️ Tecnologias
- **Linguagem:** Python 3.13
- **Framework:** Streamlit
- **Banco de Dados:** SQLite
- **Gráficos:** Plotly

## 🧠 Funcionalidades
- **Timer Pomodoro:** Ciclos de 25 minutos com barra de progresso.
- **Flashcards:** Sistema de perguntas e respostas por tópico.
- **Dashboard:** Visualização do percentual de acertos por disciplina.
- **Lógica de Revisão:** Sugestão automática de revisão (24h ou 7 dias) conforme o desempenho.

## 💻 Como rodar
1. Instale as dependências: `pip install -r requirements.txt`
2. Inicie o app: `streamlit run app.py`
